LAVENDER – NOTE TAKING WEB APPLICATION

Key Features:
- Real-time database  🔗 
- advanced note editor 📝 
- Light and Dark mode 🌓
- Infinite children documents 🌲
- Trash can & soft delete 🗑️
- Authentication 🔐 
- File upload
- File deletion
- File replacement
- Icons for each document (changes in real-time) 🌠
- Expandable sidebar ➡️🔀⬅️
- Full mobile responsiveness 📱
- Publish your note to the web 🌐
- Fully collapsable sidebar ↕️
- Landing page 🛬
- Cover image of each document 🖼️
- Recover deleted files 🔄📄

# Deployment used by `npx convex dev`
CONVEX_DEPLOYMENT=                       for database           using https://www.convex.dev/  
NEXT_PUBLIC_CONVEX_URL=                  for database           using https://www.convex.dev/  

NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=       for authentication     using https://clerk.com/       
CLERK_SECRET_KEY=                        for aythentication     using https://clerk.com/       

EDGE_STORE_ACCESS_KEY=                   for cover image        usign https://edgestore.dev/   
EDGE_STORE_SECRET_KEY=                   for cover image        using https://edgestore.dev/

npx convex dev
